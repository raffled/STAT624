Form a directory called cdh5mr2-rhipe0.75 once downloaded.

Put the examples.zip in this directory and unzip.

From the virtual machine home (cdh5mr2-rhipe0.75) run:

vagrant up

vagrant ssh virtualbox

You will be in ~ in the VM.

Execute:

cp -r /vagrant/examples ~

There are two examples---one R and one Java

There is a readme.txt file in each directory.
